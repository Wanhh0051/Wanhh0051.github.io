# 内容创作场景配置

## 内容创作到商业变现

### 阶段1: 内容策略
```yaml
主导Agent: content-creation-specialist
协作Agent:
  - product-strategy-planner (商业定位)
  - goal-management-coach (目标对齐)
质控Agent: goal-management-coach
输出物:
  - 内容主题规划
  - 目标受众分析
  - 发布平台策略
  - 商业化路径设计
```

### 阶段2: 多媒体制作
```yaml
主导Agent: multimedia-content-creator
协作Agent:
  - content-creation-specialist (内容优化)
  - ux-insight-specialist (设计审核)
质控Agent: ux-insight-specialist
输出物:
  - 文字内容
  - 图片/视频素材
  - 设计元素
  - 互动组件
```

### 阶段3: 社交运营
```yaml
主导Agent: social-media-manager
协作Agent:
  - indie-dev-marketer (增长策略)
  - content-creation-specialist (内容一致性)
质控Agent: content-creation-specialist
输出物:
  - 发布计划
  - 互动策略
  - 社群管理方案
  - 数据分析报告
```

### 阶段4: 商业变现
```yaml
主导Agent: monetization-strategist
协作Agent:
  - developer-crm-expert (客户管理)
  - product-strategy-planner (商业模式)
质控Agent: product-strategy-planner
输出物:
  - 变现策略
  - 价格体系设计
  - 客户关系管理
  - 收入优化方案
```

### 闭环机制
- 数据驱动的内容优化
- 实时策略调整
- 用户反馈收集和响应
- 持续改进和创新

## 内容类型处理

### 文章写作
```yaml
Agent组合: content-creation-specialist + multimedia-content-creator
处理流程:
  1. 主题研究和大纲设计
  2. 初稿撰写
  3. 内容优化和润色
  4. 配图和设计
  5. 发布和推广
```

### 视频制作
```yaml
Agent组合: multimedia-content-creator + social-media-manager
处理流程:
  1. 脚本策划和创意设计
  2. 拍摄和录制
  3. 后期编辑和特效
  4. 平台发布和推广
  5. 数据分析和优化
```

### 社交内容
```yaml
Agent组合: social-media-manager + content-creation-specialist
处理流程:
  1. 平台分析和策略制定
  2. 内容创意和制作
  3. 发布时机优化
  4. 互动和社群管理
  5. 效果评估和调整
```

---

**适用场景: 内容创作、社交媒体、数字营销、知识付费**
